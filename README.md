# Energy AI Simulation Platform

This project simulates a complete energy intelligence system using Python.

## System Overview

Smart meter data → forecasting → reinforcement learning → digital twin simulation

## Features

- Synthetic smart meter data generation
- Feature engineering for time series
- Energy demand forecasting
- Simple RL-based control system
- Digital twin simulation layer
- Full pipeline orchestration

## How to run

```bash
pip install -r requirements.txt
python main.py
